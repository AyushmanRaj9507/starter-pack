# 🌍 Plan With Ayushman

A beautifully designed React app that allows users to explore, search, and bookmark tour destinations with interactive UI, dark mode, and image previews.

- 🌗 Dark / Light Mode Toggle
- 🔍 Search Bar with animation
- 🖼️ Image Modal Preview
- ❤️ Bookmark "Want to Visit" feature
- 🧹 Remove or Refresh Tours
- 📱 Fully Responsive UI
